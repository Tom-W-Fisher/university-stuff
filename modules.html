Modules

This webpage is about the modules on my course.

  • Programming

    This module covers some of the basics of developing computer programs in theory and practise, covering OOP and and GUI construction. The primary language used is python, although Dart is also used later in the module.

  • Networks

    This module covers some of the basics of computer networks, including a focus on:

    • Data communication, including legacy systems;

    • Network management

    • Network Security

  • Core Computing Concepts

    This module is comprised of four sub-modules:

    • Algorithms and ethics

      This sub-module encourages students to think about the relationship between algorithms and ethics. The content, which includes a look at various philosophical approaches to morality as well as a touch of graph theory and pathfinding algorithms, builds towards a single piece of coursework, designing an ethical algorithm for a self-driving car.

    • Web

      This module introduces students to some basic web technologies: HTML (and semantic markup) and CSS styling.
